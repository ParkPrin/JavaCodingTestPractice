# 큐를 사용해서 스택을 구현하라.

큐 인스턴스를 사용해서 Stack 인터페이스를 구현하라.

    pop() 오퍼레이션은 가장 마지막으로 추가한 값을 꺼내야 한다.
    push() 오퍼레이션은 값을 추가한다.
    큐가 제공하는 offer(), poll(), isEmpty(), size()만 사용할 수 있다. Deque는 사용하지 못한다.
    (힌트) 큐 인스턴스를 여러개 사용할 수 있다.
